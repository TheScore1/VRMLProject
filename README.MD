VR Speech Trainer — Setup Guide

Этот проект использует LibreOffice, Whisper, и Ollama для работы модуля распознавания речи и генерации ответов. Ниже приведена полная инструкция по подготовке окружения.

---------------------------------------------------------------------

1. Скачать архив проекта
Скачайте архив проекта по ссылке: **[https://disk.yandex.ru/d/yas0ZUFmRYoODQ]**
Распакуйте архив.

После распаковки положите папки LibreOffice и vr_speech_trainer в корень проекта.

---------------------------------------------------------------------

2. Установка Whisper

1. Скачайте модель Whisper (~1.5 ГБ): **[https://huggingface.co/ggerganov/whisper.cpp/tree/main]** (можно любую, но тестировалась ggml-large-v3-turbo)
2. Создайте папку:
   Assets/Whisper/
3. Поместите скачанную модель в Assets/Whisper/. Если была скачана не ggml-large-v3-turbo, нужно указать путь к модели в WhisperManager (сцена с главным меню)

---------------------------------------------------------------------

3. Установка Ollama

Скачайте Ollama:
https://ollama.com

---------------------------------------------------------------------

4. Запуск Ollama в режиме сервера

Ollama должен работать в фоне, чтобы VR Speech Trainer мог к нему подключаться.

Запуск:
    ollama serve

Автоматический запуск пока не реализован, поэтому сервер нужно запускать вручную.

---------------------------------------------------------------------

5. Загрузка модели Gemma

Для работы требуется модель gemma:7b-instruct-q4_0 (~5 ГБ):

    ollama pull gemma:7b-instruct-q4_0

---------------------------------------------------------------------

6. Проверка наличия модели

Убедитесь, что модель загружена:

    ollama list

В списке должна быть модель gemma:7b-instruct-q4_0.

---------------------------------------------------------------------

7. Требование для vr_speech_trainer

Для корректной работы vr_speech_trainer должен быть запущен Ollama сервер:

    ollama serve

Без него текстовая ИИ-часть работать не будет.

---------------------------------------------------------------------

8. Как запускать vr_speech_trainer

Точка входа Python-части — main.py.

Запуск через Unity:
Если запускать проект со сцены главного меню, Unity запускает main.py автоматически.

Запуск вручную:
Если запуск идёт со сцены выступлений, Python-скрипт нужно стартовать вручную:

    python main.py
